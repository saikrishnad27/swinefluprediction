# classification_using_R
This provides classification of the data using  classifiers RandomForest, CART, Fuzzy logic, and neural network on a data set named as swineflu.csv 

CART.R -- This generates Classification and Regression Trees for the given dataset
ANN.R -- This generates a neural net for the given dataset
flexnew.R -- This provides output with a fuzzy based logic. 
randomforest.R -- this generates output with random forest algorithm.
